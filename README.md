Seidel Client
=============
A custom client for Minecraft Beta 1.7.3 adding many QoL features.

## Features
- Fullbright,
- Fog toggle,
- Adjustable FOV,
- Command window (enter commands by pressing /),
- Skin support (using [Minotar](https://minotar.net/)'s skin API)

## Upcoming features
- Gamepad support,
- Client-side commands,
- Click GUI,
- Minimap,
- Waypoints

## Known issues
- Fullbright applies after rejoining the world
